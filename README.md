# Rostov Travel - ИИ-помощник для культурного путешественника

Полнофункциональная платформа для планирования персонализированных туристических маршрутов по Ростовской области с использованием искусственного интеллекта.

## 🚀 Возможности

### Backend
- ✅ Микросервисная архитектура на Node.js
- ✅ RESTful API с полной документацией
- ✅ Интеграция с AI сервисом на Python
- ✅ Аутентификация и авторизация (JWT)
- ✅ Управление пользователями и профилями
- ✅ Генерация маршрутов с помощью AI
- ✅ Система бронирования
- ✅ Интеграция платежных систем (Stripe, ЮKassa)
- ✅ Кеширование (Redis)
- ✅ Rate limiting и безопасность
- ✅ Логирование и мониторинг
- ✅ Graceful shutdown
- ✅ Docker поддержка

### AI Service
- ✅ Генерация персонализированных маршрутов
- ✅ Чат-помощник для ответов на вопросы
- ✅ Рекомендательная система
- ✅ Оптимизация маршрутов
- ✅ Анализ предпочтений пользователей

### Frontend
- ✅ Адаптивный дизайн
- ✅ Интуитивный интерфейс
- ✅ Интеграция с картами
- ✅ Онлайн бронирование
- ✅ Личный кабинет
- ✅ Офлайн режим (PWA)

## 📋 Требования

- Node.js >= 16.0.0
- Python >= 3.9
- MongoDB >= 5.0
- Redis >= 6.0 (опционально)
- Docker и Docker Compose (для контейнеризации)
